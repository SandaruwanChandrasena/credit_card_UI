# 💳 Credit Card Form UI (React)

## 📌 Description

This project is a simple and modern **Credit Card Form UI built with React**. It allows users to enter card details with a real-time visual preview of the card.

The UI includes smooth animations, card flip effect for CVV, and custom styled dropdowns for a better user experience.

---

## ✨ Features

* Real-time card preview
* Card flip animation (CVV)
* Custom dropdown for month & year
* Clean and responsive UI

---

## 🛠️ Tech Stack

* React
* JavaScript
* CSS
